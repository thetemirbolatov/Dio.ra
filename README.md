# Dio.ra 1.0

![Dio.ra](datas/logo.ico)

**Стильный калькулятор в дизайне macOS с многоязычным интерфейсом и историей вычислений**

---

## О программе

Dio.ra — это элегантный калькулятор, вдохновленный дизайном macOS. Темный интерфейс, плавные анимации кнопок, поддержка трех языков и встроенная история вычислений делают его идеальным инструментом для повседневных расчетов.

## Возможности

### Стильный дизайн
Интерфейс в стиле macOS с округленными кнопками, оранжевыми акцентами и плавными эффектами наведения. Темная и светлая темы оформления.

### Полноценный калькулятор
Все базовые операции: сложение, вычитание, умножение, деление. Работа с процентами, смена знака числа, десятичные дроби.

### История вычислений
Все вычисления сохраняются в историю. Просматривайте предыдущие расчеты, копируйте результаты одним нажатием. История сохраняется в JSON файл.

### Многоязычный интерфейс
Приложение поддерживает три языка интерфейса:
- Русский
- Английский (English)
- Карачаевский (Къарачай)

Язык переключается в настройках и сохраняется автоматически.

### Горячие клавиши
Быстрое переключение языка без захода в настройки:
- Ctrl+R — Русский
- Ctrl+E — English
- Ctrl+K — Къарачай

### Работа с клавиатурой
Полная поддержка ввода с клавиатуры. Набирайте цифры и знаки операций не отрывая рук от клавиатуры. Backspace для удаления, Enter для расчета.

### Копирование и вставка
Выделяйте результат мышью, копируйте через Ctrl+C или контекстное меню. Вставляйте числа через Ctrl+V.

### Адаптивный дисплей
Размер шрифта автоматически уменьшается для длинных чисел. Максимальная читаемость при любых значениях.

### Сохранение настроек
Все настройки автоматически сохраняются в файл конфигурации. Сохраняются: язык интерфейса, тема оформления, состояние истории.

## Системные требования

- Windows 7, 8, 10 или 11 (32-бит или 64-бит)
- 1 ГБ оперативной памяти
- 100 МБ свободного места на диске

## Загрузка

Доступны два варианта установки:

### Установщик
Традиционная установка с ярлыками в меню Пуск и на рабочем столе.

**Файл:** `Dio.ra_Setup.exe`

Запустите установщик, следуйте инструкциям на экране. Программа установится со всеми необходимыми файлами.

### Портативная версия
Запуск без установки. Подходит для использования с USB-накопителей и внешних дисков.

**Файл:** `Dio.ra.exe`

Скачайте исполняемый файл и запустите. Установка не требуется.

## Как использовать

### Установка через установщик

1. Скачайте `Dio.ra_Setup.exe`
2. Запустите установщик
3. Следуйте указаниям мастера установки
4. Запустите Dio.ra из меню Пуск или с рабочего стола

### Использование портативной версии

1. Скачайте `Dio.ra.exe`
2. Поместите файл в любое удобное место
3. Дважды щелкните по `Dio.ra.exe`
4. Установка не требуется

Портативная версия не изменяет реестр системы и не требует прав администратора.

## Описание интерфейса

### Верхняя панель

- **Кнопка истории (слева)** — Просмотр истории вычислений
- **Кнопка настроек (справа)** — Открыть настройки приложения

### Дисплей

Отображает текущее число или результат вычисления. Поддерживает выделение текста мышью и копирование.

### Клавиатура калькулятора

- **Серые кнопки** — Очистка (C), смена знака (±), проценты (%)
- **Темные кнопки** — Цифры и десятичная запятая
- **Оранжевые кнопки** — Операции (+, −, ×, ÷) и равно (=)

---

## Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| 0-9 | Ввод цифр |
| + − * / | Операции |
| . , | Десятичная запятая |
| Enter | Вычислить |
| Backspace | Удалить символ |
| Escape | Очистить все |
| % | Проценты |
| Ctrl+C | Копировать результат |
| Ctrl+V | Вставить число |
| Ctrl+R | Русский язык |
| Ctrl+E | English language |
| Ctrl+K | Къарачай тил |

## Настройки

Откройте настройки через кнопку ⚙ в правом верхнем углу. Доступные параметры:

- **Язык** — Русский, Английский, Карачаевский
- **Оформление** — Тёмная тема, Светлая тема
- **Включить историю** — Сохранение истории вычислений

Все настройки сохраняются при нажатии кнопки "Сохранить". При смене языка приложение перезапускается.

## История вычислений

История сохраняется автоматически при каждом вычислении. Для просмотра нажмите кнопку ◷ в левом верхнем углу.

В окне истории отображаются:
- Выражение
- Результат
- Кнопка копирования результата

Историю можно очистить полностью. Максимальное количество записей — 100.
